# 目录说明

1.src目录下为托管app的页面（为了在PC上访问方便）
2.weapp目录下为天气APP程序

# 运行说明

## 主页面

### 调试

``` shell
gulp serve
```

### 构建

``` shell
gulp build
```

## 天气app

### 调试

``` shell
npm run dev
```

### 构建

``` shell
npm run build
```
